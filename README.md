## HoloVoice
Dub your videos with the same voice in just a few clicks


### Steps to use
--- 
Clone this project
```
git clone https://github.com/hazarathayya/holovoice.git
```
Go to [main.py](main.py) file in the root directory

Run this file
```
streamlit run main.py
```
